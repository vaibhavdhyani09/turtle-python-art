## What even IS this?

You know how normal people doodle in their notebooks when they're bored?

I wrote Python code to make a turtle do it for me.

This is a collection of **fun, colorful, and totally unnecessary art** drawn entirely by Python's `turtle` module — one line at a time, one shape at a time, one "woah that looks cool" moment at a time. 

No canvas. No Figma. No design degree needed.
Just a turtle, some loops, and **way too much free time.**

##  How to run it

**Step 1:** Have Python installed. (You probably do. You're here, after all.)

**Step 2:** Clone this repo like the curious soul you are —
```bash
git clone https://github.com/vaibhavdhyani09/turtle-python-art.git
cd turtle-python-art
```

**Step 3:** Pick any file and just... run it —
```bash
python your_favorite_art.py
```

**Step 4:** Watch the magic happen in real time. 



##  Fun Facts about Python Turtle

- The `turtle` module has been in Python since **forever** (literally since the 80s, inspired by Logo programming language)
- Your turtle can go **any speed** — `turtle.speed(0)` = ZOOM MODE 
- You can make it invisible. A ghost turtle. Drawing art. Spooky.
- Everything is just math pretending to be pretty
